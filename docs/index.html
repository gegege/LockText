LockText

プロジェクト概要

LockText は、テキストデータを安全に暗号化・復号できるウェブアプリケーションです。 すべての処理はブラウザ内で完結し、データやパスワードがサーバーに送信されることはありません。 機密情報の保護や安全な共有が必要な場面で、シンプルかつ強力なソリューションを提供します。

  • 完全なプライバシー保護: データとパスワードは一切サーバーに送信されません。
  • 強力な暗号化: 業界標準のAES-256-GCMアルゴリズムを使用。
  • 使いやすいインターフェース: シンプルで直感的なデザイン。
  • クロスプラットフォーム: モダンなウェブブラウザがあれば、OSを問わず利用可能。
  • オープンソース: ソースコードを公開し、透明性と信頼性を確保。

暗号化

以下の手順でテキストを暗号化します。

  • 暗号化したいテキストを入力します。
  • パスワードを入力し、確認のため再度入力します。
  • 「暗号化する」ボタンをクリックします。

復号化

以下の手順で暗号文を復号化します。

  • 復号化したい暗号文を入力します。
  • パスワードを入力します。
  • 「復号化する」ボタンをクリックします。

プライバシーポリシー

LockTextでは、ユーザーのプライバシーを保護することを最優先しています。当アプリケーションでは、入力されたデータやパスワードはサーバーに送信されず、 すべての処理がブラウザ内で完結します。暗号化や復号化の処理を行う際に、ユーザーの機密情報が第三者に漏れることはありません。
